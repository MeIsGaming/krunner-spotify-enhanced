# Configuration (EN)

Config file path:

- `~/.config/KRunner-Spotify/KRunner-Spotify.config`

## Important settings

- `COMMAND_PREFIX` (default: `spe`)
- `ENABLE_LEGACY_SP_ALIAS` (`True`/`False`)
- `CASE_SENSITIVE`
- `MAX_RESULTS`
- `VOLUME_STEP`

## Example

```ini
COMMAND_PREFIX = spe
ENABLE_LEGACY_SP_ALIAS = False
CASE_SENSITIVE = False
MAX_RESULTS = 5
```
