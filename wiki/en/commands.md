# Commands (EN)

Type `spe` in KRunner to see command autocomplete.

Legacy compatibility (optional): if `ENABLE_LEGACY_SP_ALIAS = True` is set in the
config, `sp` can also be used as prefix.

## Common examples

- `spe login`
- `spe play bohemian rhapsody`
- `spe pause`
- `spe resume`
- `spe next`
- `spe previous`
- `spe trackinfo`
- `spe setvol 50`

For the full command set, see [USAGE.md](../../USAGE.md).
