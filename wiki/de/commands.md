# Befehle (DE)

Wenn du in KRunner `spe` eingibst, siehst du Autocomplete für die verfügbaren Commands.

Legacy-Kompatibilität (optional): wenn `ENABLE_LEGACY_SP_ALIAS = True` in der
Config gesetzt ist, kann zusätzlich `sp` als Prefix genutzt werden.

## Häufige Beispiele

- `spe login`
- `spe play bohemian rhapsody`
- `spe pause`
- `spe resume`
- `spe next`
- `spe previous`
- `spe trackinfo`
- `spe setvol 50`

## Command-Übersicht (help/list)

KRunner zeigt oft nur eine begrenzte Anzahl sichtbarer Treffer gleichzeitig. Nutze diese Discovery-Shortcuts:

- `spe help`
- `spe list`
- `spe ?`
- `spe commands`
- `spe cmds`

Paging-Beispiele:

- `spe list 2`
- `spe list 3`
- `spe list next`
- `spe list prev`

Filter-Beispiele:

- `spe list vol`
- `spe help play`
- `spe commands info`

Die vollständige Befehlsliste steht in [USAGE.md](../../USAGE.md).
