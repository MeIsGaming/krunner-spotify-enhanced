# Konfiguration (DE)

Pfad der Config-Datei:

- `~/.config/KRunner-Spotify/KRunner-Spotify.config`

## Wichtige Einstellungen

- `COMMAND_PREFIX` (Standard: `spe`)
- `ENABLE_LEGACY_SP_ALIAS` (`True`/`False`)
- `CASE_SENSITIVE`
- `MAX_RESULTS`
- `VOLUME_STEP`

## Beispiel

```ini
COMMAND_PREFIX = spe
ENABLE_LEGACY_SP_ALIAS = False
CASE_SENSITIVE = False
MAX_RESULTS = 5
```
